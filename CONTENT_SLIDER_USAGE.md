# Использование ContentSlider

## Описание

ContentSlider - это новый вариант слайдера с текстом на картинке и grid сеткой 50%/50%.

## Основные отличия от HeroSlider:

1. **Grid сетка**: 2 колонки по 50% (вместо 35%/65%)
2. **Текст на изображении**: Заголовок и описание располагаются поверх изображения с градиентом
3. **Два изображения**: Показывается текущий слайд и следующий слайд одновременно

## Пример использования:

```tsx
import { ContentSlider } from "@/components/shared/Slider";

// Данные для слайдов
const slides = [
  {
    id: 1,
    title: "КУПИТЬ КАРТИНУ\nНА АУКЦИОНЕ",
    subtitle: "Сейчас идёт 5 аукционов",
    date: "",
    buttonText: "Перейти",
    buttonLink: "/auctions",
    image: "/images/slide1.jpg",
    bgColor: "#2B3E7D",
    info: "На современном рынке изобразительного искусства онлайн-аукционы становятся все более популярным способом для коллекционеров и любителей живописи приобретать значимые произведения.",
    infoTitle: ""
  },
  {
    id: 2,
    title: "ШИРОКИЙ ВЫБОР",
    subtitle: "",
    date: "",
    buttonText: "Смотреть",
    buttonLink: "/catalog",
    image: "/images/slide2.jpg",
    bgColor: "#4A5568",
    info: "У нас представлены картины самых разных художников, от всемирно известных мастеров до молодых талантов, что позволяет каждому покупателю найти что-то особенное.",
    infoTitle: ""
  }
];

export default function MyPage() {
  return (
    <ContentSlider
      slides={slides}
      minHeight="100vh"
    />
  );
}
```

## Props:

- `slides` (required): Массив объектов слайдов
- `minHeight` (optional): Минимальная высота слайдера (по умолчанию "100vh")

## Структура объекта Slide:

```typescript
type Slide = {
  id: number;
  title: string;          // Заголовок (отображается на изображении)
  subtitle: string;       // Подзаголовок (отображается на изображении)
  date: string;           // Дата (не используется в ContentSlider)
  buttonText: string;     // Текст кнопки (не используется в ContentSlider)
  buttonLink?: string;    // Ссылка кнопки (не используется в ContentSlider)
  image: string;          // URL изображения
  bgColor: string;        // Цвет фона слайдера
  info: string;           // Описание (отображается на изображении)
  infoTitle?: string;     // Заголовок описания (опционально)
};
```

## Навигация:

ContentSlider использует те же элементы навигации, что и HeroSlider:
- Стрелки влево/вправо для переключения слайдов
- Пагинация отсутствует в текущей версии

## Анимации:

Слайдер использует те же CSS анимации из `animations.css`:
- Плавное появление/исчезновение слайдов
- Направление анимации зависит от направления навигации (влево/вправо)
